# UI v6 Typography Update

- Replaced the previous Manrope font stack with Plus Jakarta Sans for a cleaner, more premium mobile-app feel.
- Added Noto Sans SC, PingFang SC, Hiragino Sans GB and Microsoft YaHei fallbacks so Chinese text renders cleanly as well.
- Reduced overly heavy font weights to improve readability on mobile screens.
- Refined heading, label, value, metadata, status chip, button and tab typography.
- Added tabular numeric rendering for quantity, price and ID-like values so inventory data aligns more professionally.
