# UI/UX v15 Search Refinement

- Inventory search now requires a full item-name match instead of partial matching.
- Removed the visible “Exact match” result label and replaced it with clearer search-result states.
- Inventory page now shows an item-not-found state when a full item-name search returns no record.
- Manage page now shows a dedicated item-not-found message below the search panel.
- Updated search placeholder and button wording to avoid confusing partial search behavior.
