# UI v7 typography hierarchy update

This version focuses on making the interface feel more orderly and consistent.

## Changes

- Rebuilt a unified typography scale for toolbar titles, page titles, section titles, card titles, form labels, hints, values, numeric values, chips, buttons, tabs, and validation messages.
- Standardized property labels across item cards and form fields with a smaller uppercase label style.
- Standardized property values across form inputs, card metadata, numeric fields, status helper cards, and preview panels.
- Reduced overly heavy font weights to make the design cleaner while keeping important values visually clear.
- Tuned line-height and letter-spacing so mobile pages are easier to scan.
- Added responsive type adjustments for smaller screens.
