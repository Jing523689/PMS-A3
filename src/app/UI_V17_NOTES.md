# UI V17 Notes - English Code Comments

This version keeps the V16 functionality and adds detailed English comments across the main implementation files.

## Commenting scope

- Added architecture and responsibility comments to the main page components.
- Added method-level comments for REST API calls, validation, search, update, delete, and UI feedback flows.
- Added field-level comments for form models, API model conversion, exact search logic, and status/featured transformations.
- Added HTML template comments to identify toolbar, search, form, card, loading, empty-state, and tab-navigation sections.
- Added SCSS comments for the main shared design-system sections.

## Comment density

The important TypeScript implementation files are commented at approximately 25%, while templates and styles include lighter section comments to keep them readable.
